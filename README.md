# STRATUS — Weather App

A real-time weather web application with a modern dark aesthetic.

## Features
- 🔍 Search any city worldwide
- 📍 Auto-detect your location (GPS)
- 🌡 Current conditions: temperature, feels like, humidity, wind, pressure, visibility
- ⏰ 24-hour hourly forecast
- 📅 7-day daily forecast with temperature range bars
- ☀️ UV index with visual scale
- 🌅 Sunrise & sunset times + daylight duration
- ☁️ Cloud cover indicator
- 🔄 Auto-refreshes every 10 minutes

## Usage
1. Open `index.html` in any modern browser
2. Type a city name and press **Search** or **Enter**
3. Or click **Locate** to use your device's GPS

## APIs Used (Free, No Key Required)
- **Open-Meteo** — Weather forecasts (https://open-meteo.com)
- **OpenStreetMap Nominatim** — Geocoding (https://nominatim.openstreetmap.org)

## No Setup Required
This is a single HTML file — no build tools, no npm, no API keys.
Just open it in a browser and it works.
